# CLI Commands

> `git status`
>
> - 현재 working tree의 상태를 보여줌 (변경사항이 있는지?)

> `git add <파일명>`
>
> - 특정 파일을 staging area에 추가
>
> `git add .`
>
> - 모든 변경사항을 staging area에 추가

> `git commit -m "<커밋 내용>"`
>
> - <커밋 내용>을 메세지로 갖는 버전을 생성
>
> `git commit`
>
> - 메세지를 입력하는 팝업 창을 열고, 이를 기반으로 버전을 생성

> `git log` 
>
> - 버전의 히스토리를 확인 (메세지 확인)
>
> `git log -stat`
>
> - 버전 히스토리의 메세지 및 포함된 파일까지 출력
