# 함수

## 기본 형식

- 파이썬의 함수 형식은 다음과 같음:

  ```python
  def 함수명(매개변수1, 매개변수2...):
      함수
  ```

- JS에서는 다음과 같이 사용자 함수를 정의

  ``` javascript
  function 함수명(매개변수1, 매개변수2...){
      함수
  }
  
  // 혹은 아래와 같이 정의하는 것도 가능
  함수명 = function (매개변수1, 매개변수2...){
      함수
  }
  ```

- 이때 함수에서 `return`을 사용하게 되면, **함수를 종료** 시키고, return 이후의 값을 반환
