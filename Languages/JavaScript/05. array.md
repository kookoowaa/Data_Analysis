# 배열

## 변수 선언

- 파이썬의 리스트와 유사한 `[]`방식으로 JS의 배열을 변수로 선언 가능

  ```javascript
  var tmp_list = ['abc', 'def', 'ghi', 'jkl', 'mno']
  ```
  
- 배열의 값은 indexing을 통해 접근 가능

  ``` javascript
  tmp_list[0]
  //'abc'
  
  tmp_list[4]
  //'mno'
  ```
  

- 이때 함수에서 `return`을 사용하게 되면, **함수를 종료** 시키고, return 이후의 값을 반환
